Início
O que é

A base de dados com todas as votações nominais ocorridas no plenário da Câmara dos Deputados na legislatura iniciada em fevereiro de 2015 é a matéria prima do modelo por trás do “Vai Passar?”. Usando uma técnica estatística chamada Random Forests, o modelo usa os resultados anteriores para calcular as probabilidades de aprovação futura de uma matéria, dadas as orientações dos líderes dos partidos mais relevantes, do governo e da oposição. O modelo simula 500 votações, combinando aleatoriamente as diferentes orientações que cada líder partidário poderia dar. A partir dessa simulação, ele elege a mais provável, considerados os resultados das votações anteriores. Com isso, é possível prever a aprovação ou rejeição de um projeto de lei, de uma emenda constitucional ou até de um pedido de impeachment. A taxa de acerto é de 89%.
